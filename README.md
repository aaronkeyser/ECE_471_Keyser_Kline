Running the project file:
Everything needed to run this program is either included in this repository, or called into the project via the provided code.
All steps of the code must be ran in sequential order for the model to be properly trained and be able to analyze the given input.

Changing the output images:
Changing the output to your own output visuals or text is fairly simple.
Under the "Test with New Images" section, at the very bottom, is the code for displaying the output.
Changing the provided image links with your own will update the program accordingly.
